OA Expenses - Configuration - Modified

Expenses Configuration lets you configure how Expenses interacts with the General Ledger.

Expenses Configuration Screen

Fields

Expenses Control

The General Ledger account code for the Expenses control account. This is the account that the credit side of claims are posted to.

Post to GL

When posting Expenses transaction batches, whether to also create an interface batch of General Ledger and Status transactions.

Default Bank

The default bank account for Expenses payments.

For each of the following, the box if you want detailed posting (individual transactions are posted to the relevant accounts); leave the box uned if you want only the batch total to be posted to the accounts.

Control

Whether to post to the Expenses control account in detail.

Whether to post in detail.

Cash Accounts

Whether to post payments in detail to the General Ledger.

Expense Accounts

Whether to post expense costs in detail.
