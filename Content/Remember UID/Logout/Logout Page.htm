Logout Page

When a user whose IP address is defined in the WEBRIPS list logs out, they are taken to the page specified by the SY parameter LOGOUT. A default page is provided, which is used if LOGOUT is blank. This default page has links to allow the user to log in to (which will bypass the login screen), or to allow them to log in as a different user (which will force them to log in using the login screen).

We anticipate that clients will create their own "log in to /logout" page, for example as part of their corporate intranet, using the links on the default page as a basis.
